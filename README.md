# 🏨 Hotel Reservations Manager

A robust web application designed to streamline hotel operations, room bookings, and guest management. Built using **ASP.NET Core MVC** and **Entity Framework Core**.

## 🚀 Key Features

* **Role-Based Access:** Distinct interfaces and permissions for **Admins**, **Employees**, and **Guests**.
* **Reservation System:** Real-time booking with automated price calculation based on service types (Breakfast, All-Inclusive).
* **Smart Validation:** Prevents over-booking by checking room capacity against guest counts.
* **Privacy-Focused:** Guests can only view their own reservations and family members they've registered.
* **Quick Guest Entry:** AJAX-powered functionality for adding family members directly from the reservation screen.

## 🛠 Tech Stack

* **Framework:** .NET 8 / ASP.NET Core MVC
* **Database:** MS SQL Server
* **ORM:** Entity Framework Core (Migrations)
* **Frontend:** Bootstrap 5, Razor Pages, JavaScript (AJAX)
* **Architecture:** N-Tier (Web, Business, Data, Models)
